Kata Vending Machine project in C# using Visual Studio 2017 Community.

You must have a Visual Studio command prompt open for the following to work.
clean.bat - Cleans the release and debug projects.
make.bat - Rebuilds the project. Release. 
test.bat - Runs the unit tests. Release.
makeD.bat - Rebuilds the project. Debug.
testD.bat - Runs the unit tests. Debug.

Most interesting files:
kata_vending\Money.cs
kata_vendingTests\MoneyTests.cs
